# Student Performance Data Pipeline

## Overview

This project demonstrates a complete ETL (Extract, Transform, Load) pipeline built using Python, Pandas, PostgreSQL, SQL, and Power BI. The pipeline processes student performance data, performs data cleaning and transformation, loads the data into PostgreSQL, and generates analytical insights through SQL queries and interactive dashboards.

## Technologies Used

* Python
* Pandas
* PostgreSQL
* SQLAlchemy
* SQL
* Power BI
* Git & GitHub

## Project Workflow

### 1. Extract

* Read student data from CSV files.
* Load data into Python using Pandas.

### 2. Transform

* Clean and preprocess data.
* Calculate average marks.
* Categorize student performance as Excellent, Good, or Needs Improvement.

### 3. Load

* Store transformed data in PostgreSQL.
* Create a structured database table for analysis.

### 4. Analyze

* Perform SQL queries to analyze student performance.
* Identify top-performing students.
* Evaluate attendance statistics.

### 5. Visualize

* Build interactive Power BI dashboards.
* Monitor performance metrics and attendance trends.
